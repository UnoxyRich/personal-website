# UnoxyRich — Personal Site

Vue 3 + Vite + TypeScript + TresJS. Pure black-and-white, animated, dark mode only.

## Develop

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deploy

Pushes to `main` auto-deploy via GitHub Actions to GitHub Pages. Enable Pages → "GitHub Actions" in repo settings.

## Edit content

All copy lives in `src/content/*.ts`. The repo-root `*.md` files are the source notes — keep them in sync if you edit them.
